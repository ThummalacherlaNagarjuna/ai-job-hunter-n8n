# AI Job Hunter Bot

An automated job hunting workflow built with n8n.

## What it does
- Reads CV from Google Drive
- Searches jobs using Remotive API
- Matches jobs using Groq AI
- Sends results to Gmail

## Tools Used
- n8n
- Groq AI (Llama 3.1)
- Remotive API
- Google Drive
- Gmail

## Setup
1. Import the workflow JSON into n8n
2. Add your API keys
3. Set your CV file ID
4. Run!
